# Contatos Specification

## Overview

The Contacts screen displays alternative support contact methods.

## UI/UX

### Layout
- List of contact cards
- Each card: icon + title + description

### Components
- **WhatsApp Card**: MessageCircle icon, "Fale pelo WhatsApp"
- **Email Card**: Mail icon, "Envie um email"
- **Phone Card**: Phone icon, "Ligue para nós"
- **Hours Card**: Clock icon, "Horário de atendimento"

### Visual Design
- Cards grid: 2 columns
- Each card: white bg, subtle shadow, icon on left

## Acceptance Criteria
- [ ] All 4 contact cards display
- [ ] Cards are clickable (future: links)
